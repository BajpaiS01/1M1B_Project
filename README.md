# AI-Driven Water Quality Prediction System

An AI-powered machine learning system developed as part of the **1M1B – IBM SkillsBuild AI + Sustainability Virtual Internship** to predict multiple water quality parameters using historical environmental data. The project applies Multi-Output Machine Learning to support environmental monitoring and promote sustainable water resource management.

## Problem Statement

Water quality plays a crucial role in protecting public health, preserving aquatic ecosystems, and ensuring sustainable development. Conventional water quality assessment methods rely on laboratory testing, which can be time-consuming and resource-intensive.

This project aims to develop an AI-driven system capable of predicting multiple water quality parameters simultaneously using historical environmental data. The predicted values can support researchers and environmental authorities in understanding water quality trends and making informed decisions.

## Dataset

The model is trained using historical water quality observations collected from multiple monitoring stations.

### Predicted Water Quality Parameters

- Dissolved Oxygen (O₂)
- Nitrate (NO₃)
- Nitrite (NO₂)
- Sulphate (SO₄)
- Phosphate (PO₄)
- Chloride (Cl)

The dataset undergoes preprocessing before model training to improve prediction performance.

## Machine Learning Model

The prediction model is implemented using:

- **MultiOutputRegressor**
- **RandomForestRegressor**

The MultiOutputRegressor enables simultaneous prediction of multiple water quality parameters from a single input instance.
